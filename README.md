# OTT User Behaviour Analysis

## Overview
Exploratory data analysis on an OTT platform user dataset to uncover viewing patterns, genre preferences, and subscription behaviour.

## Dataset
- 30 users, 10 features
- Columns: user_id, age, gender, city, device, genre, watch_hours_per_week, rating_given, subscription_type, subscription_months

## Key Findings
- Premium subscribers watch 3x more content than Basic users (15.8 hrs vs 5.2 hrs/week)
- Comedy and Action are the most popular genres among users under 25
- Strong 0.96 correlation between subscription duration and watch hours
- Documentaries underperform in both watch time and ratings
- Mobile is the preferred device among Premium subscribers

## Business Recommendations
- Invest more in Comedy and Action content to retain younger audiences
- Optimize mobile experience for Premium users
- Launch re-engagement campaigns for low watch-hour users to reduce churn
- Bundle Documentary content with popular genres to boost visibility

## Tools Used
- Python, Pandas, NumPy, Matplotlib, Jupyter Notebook
